2. 각각 호환성을 고려하여 구현할수있는 한계를 이해
3. 간단한 시각적 효과를 얼만큼 극대화 할수있는지에한 연구

1. 로딩페이지 만들기

##### CSS3 활용 STEP 2

3. 역시 개성있는 효과로 표현해볼것
