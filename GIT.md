# Git

#### Git 이란?
> 깃(Git)은 프로그램 등의 소스 코드 관리를 위한 분산 버전 관리 시스템이다.
> 빠른 수행 속도에 중점을 두고 있는 것이 특징이다.
> 최초에는 리누스 토르발스가 리눅스 커널 개발에 이용하려고 개발하였으며, 현재는 다른 곳에도 널리 사용되고 있다.
> -- 위키백과 --

